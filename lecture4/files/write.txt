std::fstreamопціонально приймає другий аргумент у своєму конструкторі — параметри відкриття файлу. Це:

std::ios::app— переходити до кінця файлу при кожній операції запису;
std::ios::binary— відкрити файл як двійковий;
std::ios::in — відкрити файл лише для читання;
std::ios::out— відкрити файл лише для запису;
std::ios::trunc— очистити файл після відкриття;
std::ios::ate— перейти в кінець файлу після відкриття (один раз).
Параметри можна комбінувати за допомогою побітового |оператора АБО ( ).
